## 执行流程

- 检查页面依赖文件状态。
- 比较旧版本依赖文件与新版本依赖文件。
- 结合页面更新要求生成差异化或全量草稿。
- 锁定或禁用页面禁止更新。

## 安全边界

- 不允许生成 Runtime YAML。
- 不允许直接修改真相文件。
- 写操作必须先生成草稿或候选结果。
- 锁定页面和禁用页面禁止自动更新和修复。
- 禁用页面默认不参与知识图谱生成。
- 发布必须由用户明确授权。
